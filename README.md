# California Multi-Temporal Remote Sensing Change Detection for Vegetation Stress Monitoring

This project explores multi-temporal vegetation changes in California using remote sensing data. The goal is to detect environmental stress patterns through time series analysis and anomaly detection.

## Project Goals

- Analyze vegetation change over time
- Detect abnormal drops in vegetation index
- Visualize temporal trends
- Build a reproducible environmental data science workflow

## Tech Stack

- Python
- Jupyter Notebook
- pandas
- numpy
- matplotlib
